この記事では、テーマが一般的な Markdown のタイポグラフィ要素をどのように表示するかを紹介します。

## テキストスタイル

通常のテキスト、**太字**、*斜体*、~~打ち消し線~~、`インラインコード`、そして[ハイパーリンク](https://example.com)。

組み合わせ: **太字と~~打ち消し線~~**、そして *斜体と`インラインコード`*。

## 引用

> 格言、抜粋、強調したい文章などの引用文。

> 複数段落の引用にも対応しています。
>
> これは同じ引用ブロック内の2つ目の段落です。

## リスト

順序なしリスト:

- りんご
- バナナ
- オレンジ
  - ネスト項目 A
  - ネスト項目 B

順序付きリスト:

1. ステップ1: 準備
2. ステップ2: 実行
3. ステップ3: 検証

## コードブロック

インラインコード: 作業ツリーを確認するには `git status` を使います。

フェンスコードブロック（Python）:

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"

print(greet("world"))
```

フェンスコードブロック（Shell）:

```shell
# 依存関係をインストール
pip install -r requirements.txt

# 開発サーバーを起動
pelican content --autoreload --listen
```

## テーブル

| 機能            | 対応  | 備考                  |
|-----------------|-------|-----------------------|
| 太字            | あり  | `**text**`            |
| 斜体            | あり  | `*text*`              |
| 打ち消し線      | あり  | `~~text~~`            |
| コードブロック  | あり  | シンタックスハイライト |
| テーブル        | あり  | GFM 拡張              |

## 区切り線

---

区切り線はセクション間に視覚的な余白を作ります。
