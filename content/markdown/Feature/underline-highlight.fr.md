title: Soulignement, emphase et surlignage
date: 2026-03-06 23:34:00
Category: Feature
Tags: Thème

Le thème propose des classes CSS `u-line` et `hl` pour le soulignement et le surlignage.

Texte normal, <span class="u-line">emphase soulignée</span>.
Voici un <span class="u-line u-line-blue">soulignement bleu</span>,
voici un <span class="u-line u-line-violet">soulignement violet</span>.

```html
<span class="u-line">emphase soulignée</span>
<span class="u-line u-line-blue">soulignement bleu</span>
<span class="u-line u-line-violet">soulignement violet</span>
```

Voici un <span class="hl hl-yellow">surlignage jaune</span>,
voici un <span class="hl hl-green">surlignage vert</span>,
voici un <span class="hl hl-pink">surlignage rose</span>.

```html
<span class="hl hl-yellow">jaune</span>
<span class="hl hl-green">vert</span>
<span class="hl hl-pink">rose</span>
<span class="hl hl-blue">bleu</span>
<span class="hl hl-lavender">lavande</span>
```

Voici un <span class="hl hl-marker">surlignage style marqueur</span>, <span class="hl hl-marker hl-green">marqueur vert</span>.
