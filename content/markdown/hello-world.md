Welcome to **notmyidea-vine**, a clean and modern theme for [Pelican](https://getpelican.com/) static site generator.

This demo site is deployed on [Cloudflare Pages](https://pages.cloudflare.com/) to showcase the theme's features.

## What is notmyidea-vine?

notmyidea-vine is a fork of notmyidea-plus, designed to be easy to deploy and customize. It features:

- Clean, responsive layout
- Code syntax highlighting
- Tag cloud support
- RSS/Atom feeds
- Giscus comments (optional)
- Analytics integrations (optional)
