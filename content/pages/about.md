## About This Site

This is the official demo site for **notmyidea-vine**, a Pelican theme built for simplicity and easy deployment.

### The Theme

notmyidea-vine is a fork of [notmyidea-plus](https://github.com/dondone-dev/notmyidea-plus), designed with:

- Clean, distraction-free reading experience
- Mobile-responsive layout
- Code syntax highlighting
- Optional Giscus comments
- Cloudflare Pages ready
